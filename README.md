# DPM Tools

Work-in-progress toolkit for digital pattern / camouflage design workflows.

## Apps

| App | Description |
|-----|-------------|
| [dpm-tool-1](./dpm-tool-1/) | Digital camo pattern generator (Electron + React). Reference-driven presets, SVG export, modular floating windows. |

## Development

Each app is self-contained. From an app directory:

```bash
cd dpm-tool-1
npm install
npm run dev
```

## Status

Experimental dev tooling — not production-ready. Expect rough edges.
